Why use AlgoRun?

AlgoRun is an Algorithm Execution Platform
It takes you less than 5 minutes to webify your algorithm and make it available online for anyone to use ..

Exposure

AlgoRun provides your algorithm with an elegant web interface that makes it easy for users to discover, understand and use your algorithm

Reusability

AlgoRun makes your algorithm immediately available for use from the web browser, through a dedicated API or through the good old command line. No more dependencies hell

Simplicity

No code change required, publishing your algorithm with AlgoRun takes less than 5 minutes
