When you deploy your algorithm on AlgoRun platform, a web interface is automatically generated for your algorithm. Users can try your algorithm directly from their web browsers. They don't need to build your source code to try it ..

Are you worried about re-writing your algorithm to run on our platform? It's no longer a problem. Keep your codebase as it is. All what AlgoRun does for you is to wrap your code along with dependencies in a container. No need to learn a new programming language nor a new software tool ..

Audience of your algorithm can run your algorithm on any machine, anytime and anywhere. Thanks to Docker containers. Users can easily download your algorithm image and run the algorithm immediately ..

AlgoRun runs on top of Docker. Not only does this mean that your algorithm will run on any machine that has Docker installed, but also means that your algorithm can reach larger number of audiences and can be deployed in applications very easily ..

Fed up with asking users to install missing packages and libraries to try your algorithm? With AlgoRun, all of your dependencies are wrapped within your algorithm container. Users will not need to install any tool other than Docker to try it ..
