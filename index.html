AlgoRun lets you easily run and publish algorithms

AlgoRun is an attempt to fix computational science reproducibility by providing computational scientists with an easy way to run and publish their algorithms. From a more technical perspective, AlgoRun aims at defining a standard docker-based packaging system to turn any command-line application into a ready-to-use web application.

With AlgoRun, docker is all what you will ever need to run and use any algorithm. No more dependencies to install, no more wrong library version... AlgoRun makes it easier than ever to run an algorithm, directly from your web browser or from a simple web API.
