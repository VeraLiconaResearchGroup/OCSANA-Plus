What is AlgoRun?

In support of the growing need in bioinformatics for easy-to-use flexible software implementations of algorithms, AlgoRun comes as a dedicated packaging system for implemented algorithms. Using Docker technology, AlgoRun supports the reproducibility of computational results by providing algorithms with a standard template container. Algorithms packaged using AlgoRun containers can be executed through a user-friendly interface directly from a web browser or via a standardized RESTful web API to allow easy integration into more complex workflows.
